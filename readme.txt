Affichez les dernières actualités du Web sur votre site Internet et les scripts, astuces et tutos de référencement et programmation.

== Description ==

p3x est un site qui vous permettra de trouver tous les outils nécessaires pour la création de votre site Web. Sur celui-ci, vous trouverez de nombreux scripts, de nombreuses astuces et de nombreux tutos pour la créer votre site Internet.

Grâce au module Wordpress p3x, vous pourrez rapidement afficher sur votre site toutes les dernières actualités informatique proposées par la plateforme p3x.
Ceci vous permettra de garnir votre contenu et de séduire vos visiteurs. En proposant plus de contenu en lien avec votre thématique, vous gagnerez surement plus de visiteurs.

== Installation ==

1. Téléchargez le plugin depuis votre administration Wordpress ou transférez le dans un nouveau dossier dans le répertoire `/wp-content/plugins/`
2. Activez le plugin depuis votre administration
3. Ajoutez vos actualités en ajoutant le tag `[p3x_actualite]` dans le contenu de vos pages
